# Predictive Sales Forecasting

## Overview

This project demonstrates sales forecasting using Machine Learning. It uses Linear Regression to predict future sales trends for both normal-growth and high-growth businesses.

## Features

* Forecasts future sales based on historical data.
* Uses Linear Regression for prediction.
* Compares:

  * Normal Business Growth
  * High Growth Business
* Generates visual sales forecast charts.
* Displays predicted sales values for future months.

## Technologies Used

* Python
* NumPy
* Matplotlib
* Scikit-learn

## Output

* Normal Business Growth Forecast Chart
* High Growth Business Forecast Chart
* Sales Forecast Comparison Chart
* Predicted sales values for future months

## Project Workflow

1. Load historical sales data.
2. Train Linear Regression models.
3. Predict future sales.
4. Visualize forecast results.
5. Compare business growth patterns.
